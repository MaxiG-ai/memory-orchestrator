# memory-orchestrator

LLM wrapper that applies memory operations. Used in my master thesis on investigating memory methods for their impact on tool use.

This project requires two config files to be set in the running repository. Examples for these can be found in this repository at `docs/model_config.example.toml` and `docs/config.example.toml`.

## Migration notes
